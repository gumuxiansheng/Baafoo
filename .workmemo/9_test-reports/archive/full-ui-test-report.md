## 问题与建议

### 🚨 P0 - 认证功能过于宽松

| 项 | 描述 |
|----|------|
| **问题** | 使用任意用户名/密码（如 `wronguser/wrongpass`）均可成功登录 |
| **预期** | 错误凭据应被拒绝，显示错误提示 |
| **影响** | 未授权用户可绕过认证访问管理功能及敏感操作 |
| **建议** | 检查认证服务配置，确保启用严格密码验证；添加登录失败次数限制 |

### ⚠️ P1 - "登录成功"提示未自动消失

| 项 | 描述 |
|----|------|
| **问题** | 登录后显示的 "登录成功" 提示消息在整个会话中持续显示 |
| **预期** | 提示消息应在 3-5 秒后自动消失 |
| **影响** | 页面视觉干扰，可能遮挡部分内容 |
| **建议** | 为 Element Plus 消息组件添加 `duration: 3000` 自动关闭配置 |
